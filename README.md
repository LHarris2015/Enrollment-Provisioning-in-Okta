<h1>Enrollment Provisioning in Okta </h1>
This tutorial outlines the enrollment and provisioning of users after importing accounts from
Microsoft Entra ID into Okta. Since imported users are initially inactive, I went through the
activation process by setting temporary passwords, logging in as users, completing enrollment
requirements, and verifying that accounts are fully functional and ready for use within the Okta
environment.<br />
<h2>Environments and Technologies Used</h2>
- Okta
<h2>Operating Systems Used </h2>
- Windows 11
